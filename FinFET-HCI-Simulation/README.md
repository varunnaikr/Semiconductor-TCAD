# FinFET Hot Carrier Injection (HCI) Simulation using Sentaurus TCAD

## Overview
This project models Hot Carrier Injection (HCI) effects in a FinFET device using Synopsys Sentaurus.

## Structure
- input/ : command + parameter files
- mesh/ : device mesh (.tdr)
- logs/ : simulation logs

## Run
smesh finfet_hci_msh.cmd
sdevice finfet_hci_des.cmd
